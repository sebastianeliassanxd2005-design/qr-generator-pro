# QR Generator Pro 🚀

Generador de códigos QR personalizados con analytics integrado. 100% gratis, sin registro, sin marcas de agua.

## 🌐 Sitio en Producción

**https://qr-generator-pro-green.vercel.app/**

## 📁 Estructura del Proyecto

```
├── index.html          # Página principal / Generador
├── features.html       # Página de características
├── analytics.html      # Dashboard de analytics
├── docs.html           # Documentación completa
├── styles.css          # Estilos compartidos
├── app.js              # Lógica del generador
├── analytics.js        # Lógica del dashboard
└── README.md           # Este archivo
```
